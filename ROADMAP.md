# GoodDay — Roadmap

Rencana pengembangan untuk **GoodDay V4** dan seterusnya.

## 🚀 Feature Ideas

- **Notifications** — Daily reminder pada waktu tertentu (Subuh / Pagi / Malam) menggunakan Web Push API atau service worker
- **Dark Mode Toggle** — Official theme switcher (Light / Dark / Auto-by-time)
- **Multiple Languages** — Dukungan bahasa Indonesia, English, dan Arab (i18n)
- **Share Feature** — Share quote ke social media (WhatsApp, Twitter/X, Instagram Stories) sebagai gambar atau teks
- **Favorite Quotes** — Bookmark & manage favorites; akses cepat ke ayat/hadits yang berkesan
- **Calendar View** — Lihat motivasi yang pernah didapat berdasarkan tanggal
- **Stats Dashboard** — Track mood patterns dari waktu ke waktu (chart, insight)
- **Backend Database** — Migrasi dari `localStorage` ke real DB (sync lintas device)
- **User Auth** — Multi-user support dengan login (email / Google / Apple)
- **AI Customization** — Personalisasi motivasi berbasis history mood & preferensi user

## 🛠 Technical Improvements

- **TypeScript** — Migrasi untuk type safety dan tooling yang lebih baik
- **Component Framework** — Refactor ke React atau Vue untuk maintainability
- **Database** — PostgreSQL atau MongoDB sebagai pengganti localStorage
- **Authentication** — JWT atau OAuth untuk multi-user
- **Testing** — Unit tests (Vitest/Jest) + E2E tests (Playwright)
- **CI/CD** — Automated deployment pipeline (GitHub Actions)
- **Analytics** — Track user behavior (privacy-respecting; e.g., Plausible)
- **Performance** — Optimize load time, lazy-load assets, image optimization

Prioritas akan disesuaikan berdasarkan feedback user dan kapasitas tim.
